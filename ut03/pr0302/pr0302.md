# Integración de servicios externos

Primero habilitamos Google images  
![alt text](<Screenshot 2025-11-25 175205-1.png>)  

Al importar los libros es importante relacionar el codigo de barras del producto cone el campo correspondiente de odoo  
![alt text](<Screenshot 2025-11-25 175434.png>)  

En la pantalla de productos, seleccionaremos los productos que queremos ponerle una imagen.  
![alt text](<Screenshot 2025-11-25 175434-1.png>)  

Ahora le daremos clic a Obtener imágenes  
![alt text](<Screenshot 2025-11-25 175623-1.png>)  

Ahora en la vista Kanban veremos los productos con la imagen correspondiente. Algunos productos se quedaran en blanco puesto que el buscador quizas no ha encontrado una imagen para el producto.  
![alt text](<Screenshot 2025-11-25 175642.png>)
