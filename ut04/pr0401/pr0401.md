## 17. Calculadora simple

```py
print("github.com/rossxrio\nSimple calc.")
def sum(x, y):
    return x + y
def sub(x, y):
    return x - y
def mult(x, y):
    return x * y
def div(x, y):
    return max(x, y) / min(x, y)

x = input('Insert the first number')
y = input('Insert the second number')
operation = input("What operation you wish to perform")

match operation:
    case '+':
        print(sum(int(x), int(y)))
    case '-':
        print(sub(int(x), int(y)))
    case '*':
        print(mult(int(x), int(y)))
    case '/':
        print(div(int(x), int(y)))
```
