## 7.- Frecuencia de palabras

```py
from functools import reduce
print("github.com/rossxrio\nWords frecuency")

def get_word_frequency(text_input):
    words = map(str.lower, text_input.split())
    return reduce(lambda d, w: d.update({w: d.get(w, 0) + 1}) or d, words, {})

text = "Hola hola mundo mundo cruel"
frequencies = get_word_frequency(text)

print(f"Input: {text}")
print(f"Frequencies: {frequencies}")

```

## 13.- Pipe de transformaciones de listas

```py
from functools import reduce
# Ejemplo de funciones y lista de entrada
funciones = [lambda x: x*2, lambda x: x+3, lambda x: x-1]
numeros = [1, 2, 3]
# Resultado esperado: [((1*2)+3)-1, ((2*2)+3)-1, ((3*2)+3)-1] -> [4, 6, 8]
a = [reduce(lambda acc, f: f(acc), funciones, n) for n in numeros]

print(a)

```
