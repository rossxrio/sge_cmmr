## 7.- Frecuencia de palabras

```py
from functools import reduce
print("github.com/rossxrio\nWords frecuency")

def get_word_frequency(text_input):
    words = map(str.lower, text_input.split())
    return reduce(lambda d, w: d.update({w: d.get(w, 0) + 1}) or d, words, {})

text = "Hola hola mundo mundo cruel"
frequencies = get_word_frequency(text)

print(f"Input: {text}")
print(f"Frequencies: {frequencies}")

```
